# Bank Management System

A console-based Bank Management System developed in C++ that allows users to manage client accounts, perform banking transactions, and apply core Object-Oriented Programming principles.

## Features
- User authentication / login system
- Add new clients
- View client list
- Update client information
- Delete client records
- Deposit money
- Withdraw money
- Organized console menus and navigation

## OOP Concepts Applied
- Encapsulation
- Abstraction
- Inheritance
- Modular class design
- Separation of responsibilities

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File handling / data storage
- Console-based interface

## Project Purpose
This project was built to practice C++ programming and strengthen understanding of Object-Oriented Programming concepts by applying them in a real-world banking system simulation.

## How to Run
1. Clone the repository
2. Open the project in your C++ IDE or editor
3. Compile and run the source file(s)
